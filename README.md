# Split Check

A tip calculator and bill splitter for iPhone. No ads, no signup, no banners.

- **Support:** https://kiddkevin00.github.io/splitcheck/
- **Privacy:** https://kiddkevin00.github.io/splitcheck/privacy.html

## Stack

Expo SDK 54, React 19.1, RN 0.81, TypeScript, expo-clipboard, expo-haptics, AsyncStorage.

## Local dev

```sh
npm install
npx expo start --tunnel
```
