Split
Check.

A tip calculator and bill splitter for iPhone. Tap, type, done — no ads, no signup, no banners chasing your wallet.

v1.0.0·iPhone · iOS 16+

01

Overview

How it works

Type the bill amount. Tap a tip percent (or a custom one). Use the + / − stepper to set how many people are splitting. Each person's share appears below in large type.

  • Copy summary puts the breakdown on your clipboard for sharing.
  • Save stores the calculation in Recent so you can recall the last ten splits.
  • Tap any Recent row to restore those values.
02

Tip presets

Why those preset percentages?

10, 15, 18, 20, 25 — the common US restaurant tipping bracket. If you're in a different region or want something else, type it in the Custom field; the preset chips become inactive.

03

Currency

What currency?

The summary uses US dollar formatting via the system locale formatter. A future version may detect device locale automatically — for now amounts are formatted as USD, but the math is currency-agnostic so it works for any currency you read the symbol as.

04

Data

What gets saved?

Only the last ten splits you explicitly tap Save on. Everything is stored on-device. No analytics, no cloud, no account.
