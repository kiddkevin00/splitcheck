Support

Frequently asked, honestly answered.

If your question isn't here, open an issue on GitHub. Public, traceable, and the same channel I use for everything else.

Does it handle other currencies?

The math is currency-agnostic, but the summary uses USD formatting via the system locale. Locale detection is on the roadmap.

Why doesn't it suggest a tip percent?

Because tipping norms vary wildly. Pick what you actually want. 18% is the default to nudge toward generous in the US bracket, nothing more.

Does it tax-split too?

Not yet — for split-by-item bills (where some people had drinks and others didn't) you'll want a different tool. Item-level split is on the roadmap.

Is anything stored?

Only the splits you explicitly tap Save on (up to 10). Everything else lives in memory until you close the app.

Get in touch

Need help, or want to report a bug?

Open an issue on GitHub. There's no support email — issues are the canonical channel and they're public, which keeps everything honest.
